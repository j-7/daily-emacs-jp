- emms
- latex
- css
- nxml
- whitespace-mode
- rainbow-delimiters
** kosmos-theme インストール
このパッケージはmelpaにあるので [[http://rubikitch.com/package-initialize][パッケージの初期設定]] を行ってください。

M-x package-refresh-contents
M-x package-install kosmos-theme
